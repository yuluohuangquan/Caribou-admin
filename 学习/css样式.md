# 3.link和@import的区别
link功能较多，可以定义RSS,定义Rel等作用，而@import只能用于加载css
link引入的css,解析到就加载，而@import引入的css要等页面完成才加载
@import需要IE5以上才能使用
link可以用js动态引入，@import不行
